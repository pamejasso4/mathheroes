El juego de matemáticas para niños (nombre) está diseñado para que pequeños estudiantes de primaria puedan aprender rápido y fácil los temas vistos con la docente Yaneth Bravo en clases, además de reforzar sus conocimientos en casa mientras juegan. 

El principal objetivo es aprender a sumar, restar, multiplicar y dividir, el juego debe contar con diversos niveles según la dificultad, el tema y el grado del alumno. Lo primero en la aplicación es el nivel, posteriormente la lección debe de ser interactiva, también si el alumno responde incorrectamente debe de marcarlo el juego, pero no eliminarlo a la primera equivocación, si no que pueda tener mínimo 2 oportunidades antes de perder. 

La aplicación será muy visual, debe de contener algunos personajes o caritas para volver más divertido el aprendizaje, también tendrá un mapa donde vendrán los niveles correspondientes a la medida que avance el jugador. Cada zona contendrá algunos incentivos, como cofres o secretos para motivar a los niños a explorar el mapa.
